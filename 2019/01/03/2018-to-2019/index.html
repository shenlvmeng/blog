2018回顾暨2019flag

发表于 2019-01-03 | 分类于 日记 |

2018

可见的进步

  • 内在素质：通用技术上有所积累，搞清楚生命和社会的来源和一些规律，个人修养上渐渐找到更自如的做人做事方式
    • 比以往更加自信
    • 看了8本能提升自己的书，有博客上的积累
      • 《设计模式之禅》
      • 《SICP》
      • 《图解HTTP》
      • 《图解密码技术》
      • 《Don’t Make Me Think》
      • 《自私的基因》
      • 《重来》
      • 《乌合之众》
  • 个人形象：有所改善
    • 开始注意穿搭和形象
      • 眼睛
      • 帽子
      • 修眉刀/眉笔
      • 帆布鞋
      • 卫衣
    • 坚持简单的锻炼身体，成效还不明显
      • 游泳
      • 无器材健身
  • 社交：认识了一些新朋友
  • 兴趣爱好：有些许投入
    • 坚持着已有的爱好
      • 拉琴
      • 骑行
      • 唱歌
    • 去了1个城市旅游
      • 厦门
  • 工作事业：适应了环境
    • 习惯了工作后的生活节奏
    • 能很好地handle交给自己的任务

可见的不足

  • 对做事和做人认识还不够透彻
  • 生活习惯不健康，健身效果不佳
  • 有还未看完的书
  • 缺少穿搭的衣物
  • 兴趣爱好上投入不够
  • 在社交上不够主动
  • 工作上积累不够，不够主动

2019

根据2018的不足，列下下面的flag：

  • 内在素质：总结出做事的规律，选择一个做人的态度，整体做人做事更自如，接受全盘的自己
    • 看完囤积的7本书，新买一些技术类书籍
    • 回顾已看的书籍，整理一个知识体系
  • 个人形象：干净会穿搭，身体健康开朗
    • 工作日坚持1点前睡觉，最晚9点半起床
    • 形成隐约的腹肌，
      • 寻找减脂方法，并控制饮食
      • 继续游泳，无器械健身
    • 补全缺失的衣物
      • 纯色棉T/长袖
      • vans/工装靴
      • 工装裤/阔腿裤/束腿裤
      • 连帽卫衣
      • 花衬衫/牛仔衣
      • 手表
      • 理发推子/吹风机
  • 社交：结识并深交新的异性朋友
  • 兴趣爱好
    • 去不止1个城市，最好去1个国家
    • 骑行：百里山水画廊/廊坊/青海湖/太湖
    • 唱歌：学3首很酷的日文歌
  • 工作事业
    • 完成晋升
    • 在技术方向上有积累和突破

祝我好运吧~

Bye~
